# Classificador KNN para Predictive Maintenance-
Projeto desenvolvido na disciplina de Aprendizado de Máquina, com o objetivo de aplicar o algoritmo K-Nearest Neighbors (KNN) em um problema real de classificação utilizando um dataset de manutenção preditiva.
## Objetivo
Construir um pipeline completo de Machine Learning que permita:
- Processar e preparar dados
- Comparar diferentes métricas de distância
- Otimizar o hiperparâmetro K
- Avaliar o desempenho do modelo
